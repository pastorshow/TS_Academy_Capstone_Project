📊 Data Directory

This folder contains all datasets used in this project.

The data structure follows standard data science best practices:

raw/ – Original datasets collected from external sources. These files remain unchanged to preserve data integrity.

processed/ – Cleaned, transformed, and merged datasets prepared for analysis and modeling.

model_data/ - model Results, Predictions and Comparison 

Depending on the project requirements, this directory may contain:

A single dataset used throughout the project, or

Multiple datasets collected from various sources and combined for comprehensive analysis.

This structure ensures reproducibility, transparency, and proper data management.
